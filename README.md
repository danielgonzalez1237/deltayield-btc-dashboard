# DeltaYield - BTC Strategy Dashboard

Dashboard interactivo para calcular la estrategia de préstamo 24h con Aave.

## Características

- 📊 Calcula el máximo préstamo que puedes pedir
- 🔄 Detalle paso a paso del looping en Aave
- 🛡️ Análisis de colchón de emergencia
- 📈 Simulador de ganancias
- 💸 Cálculo de costos

## Uso

Simplemente abre `index.html` en tu navegador o despliega en Vercel.

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
